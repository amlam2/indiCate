# Мониторинг и отображение состояния каталога входящей нормативно-справочной информации

**Название приложения:** indiCate.

**Назначение приложения:** непрерывный мониторинг состояния каталога входящей нормативно-справочной информации отделения почтовой связи а также, по необходимости, отображения файлов, находящихся в нём.

Приложение состоит из двух частей. Первая часть автоматически запускается при старте компьютера и производит непрерывное наблюдение за состоянием каталога входящей нормативно-справочной информации, оповещает пользователя в случае поступления файла или файлов, напоминает пользователю о необходимости загрузки нормативно-справочной информации а также осуществляет логирование поступивших (отправленных) файлов и действий приложения. Вторая часть запускается с помощью ярлыка на рабочем столе пользователя либо по клику на всплывающем сообщении и отображает актуальное состояния каталога входящей нормативно-справочной информации в удобочитаемом для пользователя виде. Позволяет автоматизировать рабочее место начальника отделения почтовой связи в части слежения за поступившей нормативно-справочной информацией.


Приложение написано на **Python 2.7** с использованием библиотеки кроссплатформенного графического интерфейса пользователя (GUI) **WxPython** а также набора модулей для расширенной поддержки Windows **PyWin32**.


Используются модули, не входящие в стандартную поставку интерпретатора:
        - wxPython;
        - PyWin32;
        - fsmonitor.
